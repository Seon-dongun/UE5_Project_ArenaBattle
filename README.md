# UE5_Project_ArenaBattle

- 언리얼 엔진5로 제작한 로그라이크 RPG 게임 ArenaBattle 입니다.  
- 맵을 탐험하며 적을 처치하고, 아이템 획득해 캐릭터를 성장시킬 수 있습니다.  
- 목표 단계까지 도달하면 게임을 클리어 할 수 있습니다.  
- ArenaBattle 폴더를 통해 사용된 소스 코드와 애셋을 확인할 수 있습니다

<br><br><br>
### [조작법]  
이동 : WASD / 방향키  
공격 : 마우스 좌클릭  
점프 : Space 바
시점 변환 : v
